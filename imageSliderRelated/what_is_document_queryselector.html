What is document.querySelector()?

document.querySelector() allows you to select html elements so that they can be manipulated using javascript.

Dynamically Change HTML Content

To change HTML element content dynamically using document.querySelector(), you should do 2 things:

  1. get hold or grasp the html element by its class or id and then
  2. modify the content based on some event like user click or page reload.
